## Getting Started

This is a Flutter application. To run this project:

1. Ensure you have Flutter installed ([Get Flutter](https://docs.flutter.dev/get-started/install))
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to start the app

## Project Structure

- `lib/features/` - Feature modules (social, marketplace, ride, etc.)
- `lib/core/` - Core utilities, models, and services
- `lib/services/` - Backend services and API integrations
